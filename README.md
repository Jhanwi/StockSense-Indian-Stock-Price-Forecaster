# StockSense : Indian-Stock-Price-Forecaster

**Stock Sense** is a terminal-based, object-oriented Python utility that fetches market data for Indian exchange tickers (NSE / BSE) and prints a compact, easy-to-read “stock snapshot” report in the terminal. The tool uses Yahoo Finance as a data source (via `yfinance`) and renders the output using the `rich` library for clear, colored, tabular terminal output.
